

포토샵에서는 언제나 레이어보다 선택영역의 우선순위가 높다.

그리고 레이어와 선택영역은 각각 독립적이다.

전경색 채우기 단축키 -> alt + delete
배경색 채우기 단축키 -> ctrl + delete

layer 한단계 앞으로 -> ctrl + ]
                뒤로   -> ctrl + [ 
            맨 앞으로 -> ctrl + shift + ]
            맨 뒤로   ->       "        +[

선택 영역 해제 -> ctrl + d
선택 영역 추가 -> shift 누른채로 드래그
선택 영역 빼기 -> alt 누른채로 드래그 
레이어 복사 -> ctrl + j

선택 영역 지정한뒤 ctrl + j -> 새로운 레이어에 지정된 부분만 만들어진다.
선택 반전 -> ctrl + shift +i

이미지 색상을 바꿀때 이미지에 있는 그레이 스케일 색상부분은 크게 바뀌지 않는다.(바뀌긴 함)
컬러라이즈 -> 전체 이미지 단색화

ctrl + t -> 이미지 크기 변경, 회전, 워프
